# vue-server-side-rendering
Test project to experiment with php rendered vue apps


## Setup
`# composer install`
 
`# yarn`
 
`# yarn server:build`
 
`# yarn client:build`

## Run it by
 - using the vagrant box or
 - using the php built in server `(public)# php -S localhost:8080`
 - running the webpack dev server `# yarn dev`
